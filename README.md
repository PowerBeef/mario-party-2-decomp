# Mario Party 2 (USA) Decompilation

From-scratch reverse-engineering project for Mario Party 2 (N64, `NMWE`).

## Requirements

- Python 3.11+
- A legally owned `Mario Party 2 (USA).z64` (SHA1 `166eda1c05670d337e2c3f15a5db528ae1e5d6e3`)
- Optional: `mips-linux-gnu` binutils + Linux x86 GCC 2.7.2 in `tools/gcc_2.7.2/` for byte-matching builds

## Quick Start

```bash
./install.sh
make verify
```

## Layout

- `marioparty2.yaml` — splat segment config (generated by `tools/scan_overlays.py`)
- `asm/` — disassembly output
- `src/` — matched / in-progress C
- `include/` — engine headers and structs
- `docs/` — engine documentation wiki
- `tools/` — ROM analysis utilities
